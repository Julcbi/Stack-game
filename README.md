# Standup game

This is a simple game template, Phaser framework is already implemented.

## Game Mechanics

- a very fast physics based game
- the target is a square on the right side of the screen is moving up and down
- the player shoots circles from the left bottom corner
- he clicks on the screen to load up thrust
- on mouse release the ball shoots
- if the ball hits the target, the user gets 1 point
- after 30 seconds the game is over and the score is shown
- player can try again
